COMO INSTALAR NO CELULAR:
   1. Abrir gerar-icones.html e baixar os 2 icones
   2. Colocar icones na mesma pasta dos outros arquivos
   3. Subir tudo para GitHub Pages (gratis)
   4. Abrir link no Chrome do celular
   5. Menu ... -> "Adicionar a tela inicial"

COMO IMPORTAR OS DADOS:
   1. Abrir planilha-financeira.html no navegador
   2. Clicar em "Importar Dados"
   3. Selecionar dados-importados.json (ou o .xlsx direto)
